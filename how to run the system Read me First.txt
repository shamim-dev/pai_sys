how to run the system

1st create a database in phpmyadmin name pai_system 
then import the database from the sql folder

2nd copy the pai_sys folder to htdocs of xampp
directory localdisk C/D: xampp->htdocs (Note:depends were you install your xampp)

3rd your done just run the system on a browser by this address
localhost/pai_sys

for admin  credentials for email and password use admin
